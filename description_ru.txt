Задание для кандидата (Python, django)

Реализовать микросервис на django+drf, предоставляющий http json rest ресурс, для работы с форматированной строкой.

форматированная строка: "12,323;3434,23;1,545;222,12123123"
это список пар данных (двумерный массив): "a,b;a,b;a,b;a,b"
`a`,`b`- int not null, `a` не уникально, пара не уникальна,
то есть может быть: "1,4;1,5;1,4"

Функциональность сервиса:

    1) сервис  принимает post запрос, в котором передается форматированная строка, данные сохраняются в бд

    2) принимает get запрос с параметрами фильтрации `a`, `b`  и возвращает отформатированную строку, с данными
    сгруппированными по `a` и суммой значений `b`, передача параметра фильтрации обязательна, возможна фильтрация
    по `a`и/или`b`

        a) параметр фильтра `a` отбирает те данные где `a` >= передаваемого значения
        b) параметр фильтра `b` отбирает те данные где, сумма `b` >= передаваемого значения, при группировании по `a`
        c) фильтрация по `a`, при наличии значения фильтра, первична
        d) в ответе пары отсортированы по `a`,`b`

Пример:
    передав сервису данные "3,12;1,4;1,5;1,5;3,8;5,3;5,8"
    фильтрация по `a`=4 вернет "5,11"
    фильтрация по `b`=13 вернет "1,14;3,20"
    фильтрация по `a`=3 и `b`=12 вернёт "3,20"

Пожелания к выполнению задания:
